Le plugin dico permet de demander à Sarah définition(s), synonyme(s) sur l'Internaute ou sur Wikipédia.

/!\ Avant utilisation /!\

Suite à un problème d'encodage, il est indispensable d'ajouter le dossier iconv-lite-master (téléchargeable ici) dans le dossier ...\S.A.R.A.H\NodeJS\node_modules\npm\node_modules


Les phrases magiques sont :

  • Sarah cherche * sur Wikipédia
  • Sarah cherche * dans le dictionnaire
  • Sarah donne-moi les synonymes de *
  • Sarah qu'est-ce qu'un(e) *

Si vous modifiez ces phrases, vous devrez modifier les "Mots_à_filtrer" dans le fichier dico.prop (explication ci-dessous).
/!\ Mots séparés par des espaces /!\


Explication du plugin :

Le script dico.js va tout d'abord identifier l'action qu'il devra effectuer :

  • Si "Wikipédia" est prononcé lors de la demande : il ira sur Wikipédia
  • Si "synonymes" est prononcé lors de la demande : il ira sur l'Internaute chercher le(s) synonyme(s)
  • Si aucun de ces deux mots n'est prononcé lors de la demande : il ira par défault sur l'Internaute chercher la (les) définition(s)

Ensuite, le script filtrera la demande en fonction de l'action qu'il devra effectuer :

  • Si Wikipédia : la recherche peut contenir plusieurs mots
  • Si Internaute : la recherche ne peut contenir qu'un seul mot au singulier et sans accent !

Mais pas d'inquiétude, le script s'en charge !
Il retirera tous les mots de la demande mentionnés dans les "Mots_à_filtrer" dans le fichier dico.prop pour ne garder que la recherche et vous avertira en cas de problème.
De plus, le script écrira dans la console la phrase prononcée ainsi que le sujet de la recherche obtenu après filtrage afin d'identifier facilement l'éventuelle erreur.

Exemple :
Dans la console : "Phrase prononcée : Sarah qu'est-ce qu'une voiture"
Mots_à_filtrer : "Sarah cherche sur Wikipédia dans le dictionnaire donne moi les synonymes de qu est ce un une"
Dans la console : "Sujet de la recherche : voiture"


Note :

Dans le fichier dico.xml, seul le début de la phrase est mentionné :

<item>Sarah</item>

  <one-of>
    <item>cherche</item>
    <item>donne moi les synonymes de</item>
  </one-of>

  <ruleref special="GARBAGE" />

<tag>out.action._attributes.tts="je m'en occupe";</tag>
<tag>out.action._attributes.uri="http://127.0.0.1:8080/sarah/dico";</tag>
<tag>out.action._attributes.dictation=true;</tag>

Simplement parce que special="GARBAGE" (qui permet ici d'utiliser les mots non spécifiés) ne fonctionnait pas correctement si j'y spécifiais la fin de la phrase.
